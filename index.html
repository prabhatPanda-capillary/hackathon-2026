Challenge

What to build

Use AI to solve a real problem in our implementation workflow. Your project should save time, reduce friction, or unlock a capability we don't have today.

⚡
Process automation
Identify a manual or repetitive implementation step and automate it end-to-end using AI.
🧠
Custom AI skills
Build shareable Claude skills — specialized prompts, tools, or agents — your teammates can reuse daily.
🛠️
AI-powered applications
Create a lightweight app or script that puts AI at the center of a key implementation task.
🗺️
Workflow redesign
Redesign an existing workflow from scratch with AI as the operating layer, not just a helper.

Judges will score each submission across four dimensions. Build something that's genuinely useful — that's the north star.

30%
Real-world impact
Does it meaningfully improve implementation speed, quality, or consistency?
25%
Creativity
